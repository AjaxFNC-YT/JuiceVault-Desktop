## Default Permission

Default permissions for the nativeaudio plugin

#### This default permission set includes the following:

- `allow-initialize`
- `allow-play-track`
- `allow-pause`
- `allow-resume`
- `allow-stop`
- `allow-seek`
- `allow-set-volume`
- `allow-set-eq`
- `allow-set-crossfade`
- `allow-get-state`
- `allow-dispose`
